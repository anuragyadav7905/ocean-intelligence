_train builds the Naive Bayes model from the same knowledge_base.json file.
It joins each entry's question, answer, and keywords together and runs the same cleaner used by the NLP engine.
LabelEncoder turns string categories into numbers because Naive Bayes needs numeric labels.
The model itself is a small Pipeline: TfidfVectorizer with bigrams, then MultinomialNB with alpha=0.5.
